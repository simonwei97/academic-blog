背包问题（[Knapsack problem](https://en.wikipedia.org/wiki/Knapsack_problem)）：有 $N$ 种物品，第 $i$ 种物品的重量为 $w_i$​，价值为
$v_i$​，背包能承受的最大重量为 $W$。将哪些物品装入背包，可使这些物品的总重量不超过背包容量，且价值总和最大？
