## 描述 (Description)

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j、i != k 且 j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请

你返回所有和为 `0` 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

https://leetcode.com/problems/3sum/description/

**Example 1:**

> Input: nums = [-1,0,1,2,-1,-4]

> Output: [[-1,-1,2],[-1,0,1]]

**Example 2:**

> Input: nums = [0,1,1]

> Output: []

**Example 3:**

> Input: nums = [0,0,0]

> Output: [[0,0,0]]

## 提示 (Constraints)

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

## 分析 (Analysis)

排序+双指针

## 代码 (Code)

- 时间复杂度：$O(n^2)$
- 空间复杂度：$O(1)$

```golang
func threeSum(nums []int) [][]int {
    if len(nums) < 3 {
        return nil
    }
    sort.Ints(nums)
    res := make([][]int, 0)

    for i := 0; i < len(nums)-2; i++ {
        n1 := nums[i]
        if n1 > 0 {
            break
        }

        // 去重
        if i > 0 && n1 == nums[i-1] {
            continue
        }

        l, r := i+1, len(nums)-1
        for l < r {
            n2, n3 := nums[l], nums[r]
            sum := n1 + n2 + n3
            if sum < 0 {
                l++
            } else if sum > 0 {
                r--
            } else {
                res = append(res, []int{n1, n2, n3})
                for l < r && n2 == nums[l] {
                    l++
                }
                for l < r && n3 == nums[r] {
                    r--
                }
            }
        }
    }

    return res
}
```

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        res = []
        if not nums or n < 3:
            return []
        nums.sort()
        res = []
        for i in range(n):
            if nums[i] > 0:
                return res
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            L = i + 1
            R = n - 1
            while L < R:
                if nums[i] + nums[L] + nums[R] == 0:
                    res.append([nums[i], nums[L], nums[R]])
                    while L < R and nums[L] == nums[L + 1]:
                        L = L + 1
                    while L < R and nums[R] == nums[R - 1]:
                        R = R - 1
                    L = L + 1
                    R = R - 1
                elif nums[i] + nums[L] + nums[R] > 0:
                    R = R - 1
                else:
                    L = L + 1
        return res
```
